Purpose

The bootstrap-modal-carousel.js library is made to display a carousel in a fullscreen modal box. The slides fit automatically.

The script depends on bootstrap-modal-fullscreen, bootstrap-modal-local and bootstrap-carousel-fit libraries.

Demo

{{> partials/carousel data.modal-carousel.carousels.default}}

Usage

  1. Add the class name .modal-fullscreen and .force-fullscreen (to place header and footer away) to the modal box.
  2. Add the class name .carousel-fit to the carousel.
  3. On the controller that will open the modal, add the attribute data-local="#myCarousel" to target the carousel.
