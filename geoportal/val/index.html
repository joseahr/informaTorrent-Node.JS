Què t'oferim?

Des del menú podràs accedir als diferents serveis web elaborats per l'ajuntament de Torrent seguint els estàndards aprovats pel Open Geospatial Consortium (OGC) per al seu ús en sistemes d'informació geogràfica (SIG). Seguint aquests estàndards s'intenta aconseguir la integració i la interoperabilitat de grans volums de dades espacialment referenciats (Informació Geogràfica).

Si ho desitja també pots fer ús del nostre visor online o buscar els recursos del seu interès accedint al catàleg de serveis.

Servei WMS


Els serveis web de mapes WMS (Web Map Service) produeixen mapes (imatges digitals georeferenciades) a partir de dades georeferenciades.

Servei WFS


Web Feature Service o WFS del Consorci Open Geospatial Consortium o OGC és un servei estàndard, que ofereix una interface de comunicació que permet interactuar amb els mapes servits per l'estàndard WMS, com per exemple, editar la imatge que ens ofereix el servei WMS o analitzar la imatge seguint criteris geogràfics.Per realitzar aquestes operacions s'utilitza el llenguatge GML que deriva de l'XML, que és l'estàndard a través del qual es transmeten les ordres WFS. WFS no transaccional permet fer consultes i recuperació d'elements geogràfics. Per contra WFS-T (Web Feature Service Transactional) permet a més la creació, eliminació i actualització d'aquests elements geogràfics del mapa.

Servei WCS

La interface estàndard Web Coverage Service (WCS) o Servei de Cobertura web d'Open Geospatial Consortium proporciona una interface que permet realitzar peticions de cobertura geogràfica a través del web utilitzant trucades independents de la plataforma. Les cobertures són objectes (o imatges) en una àrea geogràfica mentre que la interface WMS o portals de mapes en línia com Google Maps retornen només una imatge, que els usuaris no poden editar o analitzar espacialment.

Servei WMS Tesel·lat

Els Serveis Web de Mapes teselados WMS (Web Map Tile Service) és un protocol estàndard de l'OGC per servir tessel·les de mapa pregenerades al servidor.

Servei CSW

Els serveis CSW o Serveis de Catàleg satisfan un estàndard dissenyat per l'Open Geospatial Consortium, el qual defineix una interface comuna per al descobriment, recerca i consulta de metadades relacionats a dades, serveis i recursos de tipus geogràfic.
